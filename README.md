### 关于spring cloud的一些学习demo

#### 模块说明：  
>eureka: 注册服务  
>eureka1: 注册服务
>servera: 模拟提供服务端  
>serverb: 模拟客户服务端(使用robbin)  
>serverc: 模拟客户服务端(使用feign)  
>serverd: 模拟客户服务端断路器(robbin + hystrix)  
>servere: 模拟客户服务端断路器(feign feign中以集成hystrix)  
>serverf: 模拟注册服务  
>serverg: 模拟调用config-server的客户服务端
>serverg: 模拟zuul网关
